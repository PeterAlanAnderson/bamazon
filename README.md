# bamazon

This is a sample inventory tracker and inventory checker application for command line written in js.  The bamazonCustomer file allows you to "purchase" goods after viewing them, and alter the inventory of a database.  The bamazonManager file allows you to increase the inventory, view low inventory items, and add new items to the inventory.

### prerequisites

You'll need to set up a database using the supplied schema, as well as install inquirer and mysql.  You'll also need to update the connection details for the DB connection to meet your own credentials.
